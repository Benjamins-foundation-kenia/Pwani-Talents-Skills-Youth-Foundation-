# Pwani Talents & Skills Youth Foundation Website

A responsive, high-impact static website built to showcase the initiatives, programs, and community impact of the Pwani Talents & Skills Youth Foundation based in Mtongwe Midodoni, Mombasa, Kenya[cite: 1]. 

The platform is designed to drive local engagement, coordinate international volunteers, and facilitate community support through transparent program listings and direct communication channels[cite: 1].

---

## 🚀 Features

- **Responsive Landing Page (`index.html`)**: Features an engaging hero section, organizational pillars, and clear calls to action[cite: 1].
- **Program Showcase (`programs.html`)**: Detailed layouts highlighting the core community pillars:
  - **Talent & Vocational Skills**: Photography, digital arts, sports excellence, and tech literacy[cite: 1].
  - **Women's Empowerment**: Leadership mentorship and micro-business support[cite: 1].
  - **Sama Gardens Initiative**: Climate-smart agriculture and community gardening[cite: 1].
- **Modern UI Styling**: Built entirely with utility-first Tailwind CSS for lightning-fast performance and clean typography (Inter & Poppins).
- **Direct M-Pesa Integration**: Clear local donation pathways utilizing the foundation's official contact identifier[cite: 1].
- **Instant WhatsApp Communication**: Pre-configured dynamic chat links (`https://wa.me/...`) for instant enrollment and partnerships[cite: 1].
